# Portafolio Web

Este proyecto corresponde a un portafolio web personal desarrollado con HTML5 y CSS3, donde se presenta información sobre mí, mis habilidades, algunos proyectos y un formulario de contacto. La página fue organizada utilizando etiquetas semánticas y estilos con Flexbox y Grid para distribuir el contenido, además de incluir diseño responsive para adaptarse a distintos dispositivos como computador, tablet y celular.
